
# TryHackMe — Public Key Cryptography Basics

Finished this room today.

Went in thinking I had a decent understanding of public key cryptography and came out realizing I only had half the picture.

This room covers the fundamentals of:

* RSA
* Diffie-Hellman
* SSH keys
* PGP/GPG

What I liked most is that it’s hands-on. You actually use the concepts instead of just reading theory.

The GPG task was especially interesting. I had to decrypt a file using a private key and learned that simply having the key file on disk isn’t enough — you still need to import it into GPG’s keyring before it becomes usable. Small detail, but it made a lot of things click for me.

The SSH key section also connected nicely with tools I already use, so seeing the same public/private key concepts applied there helped tie everything together.

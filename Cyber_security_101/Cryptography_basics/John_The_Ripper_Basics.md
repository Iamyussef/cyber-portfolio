# John the Ripper Basics

John the Ripper is a password cracking tool that works by running a wordlist against a hash
until it finds a match. The room covers identifying hash types and cracking them using
rockyou.txt. The main thing to get comfortable with is specifying the right `--format` flag —
John can handle a lot of hash types but you need to point it in the right direction. It also
covers the unshadow workflow, which combines `/etc/passwd` and `/etc/shadow` into a single file
John can actually work with. Overall, a solid intro to offline password cracking and how easily
weak passwords fall.

# TryHackMe - Windows Fundamentals Part 1

## Overview
In this room, I learned the basic structure of the Windows operating system and explored some of the tools and features commonly used in Windows environments. Since Windows is widely used in enterprise environments, understanding these fundamentals is important for anyone starting in cybersecurity.

---

## What I Learned

- The differences between Windows editions
- How to navigate the Windows GUI
- The basics of the NTFS file system and permissions
- The role of administrator and standard user accounts
- How User Account Control (UAC) helps secure the system
- The importance of directories like:

```text
C:\Windows\System32
```

- Basic system monitoring using Task Manager:

```text
Ctrl + Shift + Esc
```

---

## Takeaways

This room gave me a better understanding of how Windows systems are organized and managed. I also learned how permissions and user privileges work, which are important concepts in cybersecurity.
